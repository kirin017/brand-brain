# Outputs

Thư mục này dành cho đầu ra do người hoặc AI agent tạo trong giai đoạn thử nghiệm:

- Brief thiết kế
- Caption
- Kịch bản sale
- Bài Zalo
- Kết quả brand check
- Draft campaign

Không xem `outputs/` là nguồn sự thật. Nếu một quyết định được duyệt, hãy cập nhật ngược về `brand-brain/` hoặc `brand-data/`.
