# Scripts

Thư mục này dành cho script kiểm tra, chuyển đổi hoặc automation trong giai đoạn sau.

Hiện chưa thêm script vì repo chưa kết nối API và chưa xây app.

Ví dụ script có thể thêm sau:

- Kiểm tra JSON hợp lệ
- Validate output theo schema
- Xuất dữ liệu brand-data sang prompt bundle
- Chuẩn hóa tên file campaign
