# Quy Ước Đặt Tên

## Folder

Dùng `kebab-case`:

- `brand-brain`
- `brand-data`
- `app-later`

## Markdown

Dùng `kebab-case.md`. File Brand Brain dùng số thứ tự hai chữ số:

- `01-brand-dna.md`
- `03-product-membership-brain.md`
- `12-agent-operating-brain.md`

## JSON

- Tên file: `kebab-case.json`
- Tên field: `camelCase`
- ID: `kebab-case`, không dấu

Ví dụ:

```json
{
  "segmentId": "phu-nu-ban-ron",
  "displayName": "Phụ nữ bận rộn",
  "primaryNeed": "Ăn uống lành hơn nhưng vẫn tiện"
}
```

## Prompt

Prompt agent dùng format:

```text
[vai-tro]-agent.md
```

Ví dụ:

- `zalo-community-agent.md`
- `membership-campaign-agent.md`
- `connected-point-agent.md`

## Ngày Tháng

Dùng format ISO:

```text
YYYY-MM-DD
```
