# Templates

Thư mục này dành cho template có thể tái sử dụng:

- Template brief thiết kế
- Template lịch nội dung
- Template kịch bản sale
- Template campaign membership
- Template tài liệu điểm bán
- Template kiểm tra thương hiệu

Hiện repo ưu tiên cấu trúc và prompt. Template chi tiết có thể bổ sung sau khi founder xác nhận quy trình.
