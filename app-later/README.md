# App Later

Thư mục này giữ chỗ cho app hoặc giao diện quản trị sau này.

Hiện không xây web app. Khi bước sang giai đoạn app, cần xác định:

- Người dùng chính của app
- Luồng nhập liệu Brand Brain
- Luồng tạo brief/design
- Quyền truy cập dữ liệu
- Cách validate schema
- Cách lưu output và lịch sử duyệt
